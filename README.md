# Faktor

Faktor is a Solana smart contract for issuing "Venmo-style" invoices to other parties on-chain.

### Getting started

Clone the repo and build:

```sh
git clone https://github.com/ampersand-xyz/faktor
cd faktor
cargo build
```

Run tests:

```sh
anchor test
```

Start the frontend:

```
cd app
yarn
yarn start
```
